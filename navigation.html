🪄 关于投影打印机的规定

投影打印机（Litematica Printer）是投影Mod的一个拓展，能让玩家通过在投影中放置正确的方块来更快地建造。

本服务器允许使用在Litematica Printer (modrinth.com)下载的mod，不允许使用其他魔改的打印机，比如外挂端的打印机。且速度不得超过以下规定：

  • printingInterval（打印速度 ）不低于2 （实测速度每秒钟放置不超过10个方块，每个方块之间放置的间隔不低于0.08秒）
  • printingRange（打印最大距离）不超过 3

🍺酿酒 BreweryX

游戏中玩法，未成年人禁止饮酒

BreweryX 带来前所未有的醉酒体验！​​ 通过复杂的酿造工艺，你可以制作出各种带有特殊效果的酒类饮品，在Minecraft中体验真实的醉酒感受。从发酵、蒸馏到橡木桶陈酿，每个步骤都影响着酒的品质和效果。高品质的酒能带来增益，而劣质酒则可能导致宿醉等负面效果。你可以创造自己的独家配方，举办热闹的酒会，或是经营酒馆生意。喝醉后玩家会步履蹒跚、聊天内容变得混乱，甚至会在/home醒来。快来掌握这门酿造艺术，让你的服务器充满美酒与欢笑吧！

与原版Minecraft不同，这里不是简单地将原料加入酿造台等待完成。根据配方不同，整个酿造过程可能既复杂又耗时。某些配方对原料或其他步骤要求极高精度，任何一步的失误都可能导致药水品质下降，产生劣质酒精或其他有毒副作用。

这种高难度酿造机制的意义在于，玩家至少对那些高价值药水（如朗姆酒）的配方不得而知。这使得高品质药水价值倍增，同时鼓励玩家不断尝试和完善酿造工艺。

常见步骤：

根据配方不同，有些步骤可以省略。以下是标准酿造流程：

​​发酵阶段​​

第一步是对新鲜原料进行发酵：

  1. 将炼药锅置于火源上方
  2. 倒入清水
  3. 右键添加原料
  4. 等待数分钟发酵
  5. 用玻璃瓶装瓶

使用时钟右键点击炼药锅可查看发酵时长。

​​蒸馏阶段​​

部分配方无需蒸馏：

  1. 将发酵液放入酿造台
  2. 顶部放入荧石作为过滤器（不会被消耗）
​​陈酿阶段​​

陈酿需要一个桶。

Minecraft 桶 |6 插槽
原版桶 可用于一些啤酒的陈酿。 只需制作并放置桶，然后将啤酒放入其中。

你也可以建造更大的桶。它们可以通过两种方式构建：

小桶 |9 插槽
使用 8 个木制楼梯构建桶形。
在右下角放置一个标志，并在上行写下 "Barrel"
如果成功，会有桶建造成功的提示。

大桶 |27 个插槽
使用 5 个栅栏、16 个木楼梯和 18 个木板来构建桶形。 附上一个水龙头（栅栏）在上方放一个告示牌并在第一行写上"Barrel"

如果成功，会有桶建造成功的提示。
在大桶创建成功后，你就可以拆掉告示牌了。
右击桶就能打开界面。
将酒瓶放入木桶后，每过1个游戏日相当于陈酿1年。不同木材建造的木桶会影响酒的品质（原版木桶默认为橡木）。

​​饮酒效果​​

酒精度数会影响玩家：

  • 行走不稳，难以直线移动
  • 可能触发失明、混乱、中毒等效果
  • 醉酒程度会影响聊天内容，语句可能变得语无伦次
  • 高度酒可能有毒副作用
  • 过量饮酒可能导致呕吐
  • 醉酒状态下线可能导致短时间内无法登录
  • 严重醉酒会直接断线
​​酒品密封（用于商店插件）​​

为防止交易时酒品被篡改，可使用密封台：

  • 合成配方：4木板+2玻璃瓶（1.14+版本）
  • 密封后的酒品将无法继续陈酿，且不显示详细信息
​​醒酒机制​​

酒精会随时间自然代谢，也可通过食用配置文件指定的 Recovery 食物加速清醒。极度醉酒时：

  • 短时间内重连可能出现在随机地点
  • 数小时或次日登录会在/home苏醒
  • 劣质酒会导致宿醉（缓慢+饥饿）

【/resident set perm】 - 个人权限设定

/resident set perm [某些人] [某权限] [on/off] - 指定"某些人"的"某权限"，on 为开，off 为关。

"某些人"的可选项：[resident/ally/outsider] 三个选项分别为"本城镇居民"、"盟友"、"无关人员"

"某权限"的可选项：[build/destroy/switch/itemuse] 四个选项对应的权限分别为"建造"、"破坏"、"开关使用"、"物品使用"

【/resident set mode】 - 个人模式设定

/resident set mode clear - 清除已设置的模式

/resident set mode [模式名称]

可选模式：map - 自动字符地图模式，开启后移动时聊天面板会即时更新你的地图信息、所在的位置；

可选模式：townclaim - 城镇圈地模式，开启后移动会自动圈你脚下路过的所有地区；

可选模式：townunclaim - 反圈地模式，开启后移动时会自动将你脚下所有属于你的领地删除！慎用！！！

【/town set perm】 - 国家权限类指令

/town set perm [某些人] [某权限] [on/off] - 指定"某些人"在城镇中的"某权限"，on 为开，off 为关。

"某些人"的可选项：[resident/ally/outsider] 三个选项分别为"本城镇居民"、"盟友"、"无关人员"

"某权限"的可选项：[build/destroy/switch/itemuse] 四个选项对应的权限分别为"建造"、"破坏"、"开关使用"、"物品使用"
